# robworks-code/homebrew-tap

Homebrew tap for [Robworks-Code](https://github.com/Robworks-Code) projects.

## Install

```sh
brew install robworks-code/tap/<formula>
```

## Available formulas

- [`ccmcp`](https://github.com/ringo380/ccmcp) — Dynamic CLI + TUI for managing Claude Code MCP servers, plugins, and per-project overrides.

## Maintenance

Formulas are auto-updated by [goreleaser](https://goreleaser.com) on each upstream release tag. Manual edits to `Formula/*.rb` will be overwritten.
